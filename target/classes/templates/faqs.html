FAQ's

  1. ¿Qué es el web hosting?

    El web hosting es un servicio que permite a individuos y organizaciones publicar un sitio web o página web en Internet. Un proveedor de web hosting, o web host, es una empresa que proporciona las tecnologías y servicios necesarios para que el sitio web sea accesible en Internet.

  2. ¿Qué tipos de hosting ofrecemos?

    Ofrecemos varios tipos de hosting, incluyendo hosting compartido, VPS (servidor privado virtual), hosting dedicado, y hosting en la nube. Cada uno tiene diferentes niveles de recursos y capacidades, adecuados para diferentes necesidades.

  3. ¿Puedo registrar nombres de dominio con ustedes?

    No, ofrecemos servicios de registro de solo de nuestro dominio.

  4. ¿Qué medidas de seguridad implementan?

    Utilizamos una variedad de medidas de seguridad para proteger nuestros servidores y su contenido. Esto incluye firewalls, monitoreo de seguridad las 24 horas, y certificados SSL para asegurar la transmisión de datos.

  5. ¿Ofrecen certificados SSL?

    No, al usar nuestros servicios de hosting ya incorporan eso certificados para asegurar la conexión entre los sitios web, sus visitantes y la trasferencia de archivos.

  6. ¿Qué tipo de soporte técnico ofrecen?

    Ofrecemos soporte técnico 24/7 a través de chat en vivo, teléfono y correo electrónico. Nuestro equipo está aquí para ayudarte con cualquier problema técnico o pregunta que puedas tener.

  7. ¿Tienen algún límite de ancho de banda o almacenamiento?

    Nuestros planes de hosting varían en términos de límites de almacenamiento y ancho de banda. Detallamos claramente estos límites en la descripción de cada plan para que puedas elegir el que mejor se adapte a tus necesidades.

  8. ¿Puedo instalar aplicaciones en mi hosting?

    No, deberas usar nuestra API para realizar todas tus peticiones.

  9. ¿Qué políticas de respaldo tienen?

    Realizamos copias de seguridad diarias de todos los sitios web alojados con nosotros. Estas copias de seguridad están disponibles para su restauración bajo petición.
